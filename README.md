## Descripción

Implementación en Python de algoritmos fundamentales de criptografía clásica desarrollados en Jupyter Notebook.  
El proyecto integra conceptos históricos y prácticos mediante la programación de conversiones binarias y cifrados por sustitución.

---

## Estructura del Proyecto

- `Parte1.ipynb`  
  Investigación e implementación del Cifrado de César.

- `Parte2.ipynb`  
  Implementación de:
  - Conversión Base64 → Binario → ASCII  
  - Operación XOR sobre cadenas binarias  

---

## Funcionalidades

### Conversión Base64 a ASCII
Transformación completa de un mensaje codificado en Base64 hacia su representación ASCII utilizando conversión binaria intermedia.

### Operación XOR
Aplicación bit a bit de la operación lógica XOR entre dos cadenas binarias de igual longitud.

### Cifrado de César
Implementación de cifrado y descifrado mediante desplazamiento modular sobre el alfabeto.

## Requisitos

- Python 3.x  
- Jupyter Notebook

## Ejecución

1. Abrir Jupyter Notebook.
2. Cargar `Parte1.ipynb` o `Parte2.ipynb`.
3. Ejecutar las celdas en orden.
4. Ingresar los valores solicitados en las entradas interactivas.

---

## Propósito Académico

Comprender los fundamentos de la criptografía clásica, incluyendo sustitución alfabética, operaciones binarias y aritmética modular mediante su implementación programada.
